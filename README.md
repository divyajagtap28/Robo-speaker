# Robo Speaker – Python Voice Assistant

A simple Windows-safe Python voice assistant that converts text into speech.  
It can respond to keywords like **time**, **date**, **greetings**, and **jokes**, and logs all user interactions.

## **Features**

- Text-to-speech using **pyttsx3** (Windows-safe)
- Keyword-based responses:
  - **Time** → Announces current time
  - **Date** → Announces today’s date
  - **Jokes** → Tells a random joke
- Repeats user input if no keyword is detected
- Logs all user inputs with timestamps in `log.txt`
- Simple, beginner-friendly, and fully functional
